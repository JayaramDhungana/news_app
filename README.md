# Flutter News App
A beautiful and minimal Flutter app that fetches the latest news from various categories and displays them using a clean UI.

## 🚀 Features

- Fetch  news from News API(newsapi.org)
- Dark and Light mode support
- Clean and responsive UI
- Category-wise filtering
- Save news and remove from saves
